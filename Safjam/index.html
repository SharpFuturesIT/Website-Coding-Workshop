Previous Projects

Here are websites we have designed and developed. These projects helped us grow our skills in UX, UI design, and web development.

Slovak Charity

We designed and built a charity website that displays events and includes a donation feature. It was aimed at people interested in charity events.

We created the website using HTML, CSS, and JavaScript, with an index page, donate page, and contact form.

We used JavaScript's fetch() to dynamically load a reusable navigation bar component.
We applied CSS through an external stylesheet to manage layout, colors, and typography.
We followed basic accessibility principles using alt text and semantic tags such as <main>, <section>, and <footer>.
